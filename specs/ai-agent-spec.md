# Agent 规格

| Agent | 输入 | 输出 | 失败兜底 |
|---|---|---|---|
| Diagnosis | 岗位、工具经验、三轮对话 | AI Level、优势、弱项、节奏 | 本地规则诊断 |
| Path Planner | 画像、岗位、课程库、进度 | 优先任务、节奏、解释、风险 | 确定性规则排序 |
| Companion | 当前阶段、完成任务、消息 | 提醒、解释、下一任务 | 本地知识回复 |
| Assessment | 阶段、岗位、回答或作品 | 分数、维度、短板、补强 | 保存提交，等待重评 |
| HR Insight | 聚合学习事件与评估 | 风险、Champion、课程建议 | 静态演示数据 |

所有 LLM 输出进入业务状态前必须进行结构校验；低置信结果不得直接作为绩效结论。
